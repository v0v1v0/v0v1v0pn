NAEPprimer-package R Documentation

The NAEP Primer package contains a random mini-sample of real data from the 2005 grade 8 mathematics assessment that have been approved for public use. This includes two data files, as well as a system control file that corresponds to each data file to read in and format for further processing.

Description

Most of the data is in M36NC2PM.dat (including student, teacher, and most school variables) while M36NT2PM.dat with two school variables in it for demonstration. M36NC2PM.fr2 and M36NT2PM.fr2 are format files used to to read in and format for further processing.
