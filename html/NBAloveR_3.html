getMatchups R Documentation

Season Matchups

Description

Season Matchups and results of a given team

Usage

getMatchups(team_code, season)

Arguments

team_code

Team code consisting 3 characters
season

Season number

Value

This function returns data.frame including columns:

  • G

  • Date

  • StartTime

  • Opponent

  • Result

  • OT

  • Tm

  • Opp

  • W

  • L

  • Streak

Author(s)

Koki Ando

See Also

https://www.basketball-reference.com/teams/

Examples

## Not run: 
 bos19 <- getMatchups(team_code = "bos", season = 2019)
 head(bos19)

## End(Not run)
