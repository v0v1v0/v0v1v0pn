copy_ucrt_group

Description

copy_ucrt_group

Usage

copy_ucrt_group(data_list, outname, site, file, species)

Arguments

data_list

List of groups to retrieve ucrt data from.
outname

Output file name.
site

NEON 4-letter site code.
file

Input file name.
species

H2O or CO2.

Value

Nothing to the workspace, but copies ucrt group from input file to output file.
