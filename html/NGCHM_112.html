chmLabel<- R Documentation

Set the label/name of an NG-CHM object

Description

Set the label/name of an NG-CHM object

Usage

chmLabel(x) <- value

Arguments

x

The NG-CHM object on which to set the label/name.
value

The new name (a single character string).

Value

The modified NG-CHM object.

See Also

chmLabel

Examples

hm <- chmNew("Old name")
chmLabel(hm) <- "A new name"
