NISTpoundalTOnewton R Documentation

Convert poundal to newton

Description

NISTpoundalTOnewton converts from poundal to newton (N)

Usage

NISTpoundalTOnewton(poundal)

Arguments

poundal

poundal

Value

newton (N)
