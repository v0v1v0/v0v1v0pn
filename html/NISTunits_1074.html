NISTradianTOdeg R Documentation

Convert radian to degree

Description

NISTradianTOdeg converts from radian (rad) to degree (angle) ()

Usage

NISTradianTOdeg(radian)

Arguments

radian

radian (rad)

Value

degree (angle) ()
