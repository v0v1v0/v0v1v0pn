NISTstathenryTOhenry R Documentation

Convert stathenry to henry

Description

NISTstathenryTOhenry converts from stathenry to henry (H)

Usage

NISTstathenryTOhenry(stathenry)

Arguments

stathenry

stathenry

Value

henry (H)

Jose Gama
