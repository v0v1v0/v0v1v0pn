NISTbecquerelTOcurie R Documentation

Convert becquerel to curie

Description

NISTbecquerelTOcurie converts from becquerel (Bq) to curie (Ci)

Usage

NISTbecquerelTOcurie(becquerel)

Arguments

becquerel

becquerel (Bq)

Value

curie (Ci)

Author(s)

Jose Gama

Source
