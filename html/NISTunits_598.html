NISTgonTOdeg R Documentation

Convert gon to degree

Description

NISTgonTOdeg converts from gon (also called grade) (gon) to degree (angle) ()

Usage

NISTgonTOdeg(gon)

Arguments

gon

gon (also called grade) (gon)

Value

degree (angle) ()

Jose Gama
