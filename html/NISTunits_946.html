NISTohmTOstatohm R Documentation

Convert ohm to statohm

Description

NISTohmTOstatohm converts from ohm (ohm) to statohm

Usage

NISTohmTOstatohm(ohm)

Arguments

ohm

ohm (ohm)

Value

statohm
