binom.confint

Description

Exact confidence intervals for a binomial proportion parameter.

Usage

binom.confint(k, n, alpha = 0.05, side = c("two", "upper", "lower"))

Arguments

k

#"heads"
n

sample size
alpha

Confidence level
side

Sidedness of the hypothesis: c("two", "upper", "lower")
