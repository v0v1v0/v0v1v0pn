FourSources R Documentation

Four sources

Description

This data set is used by Coolen and van der Laan (2001) to introduce NPI for multiple comparisons.

Usage

data(FourSources)

Format

An object of class "data.frame"

source

Source A, B, C or D

value

values given for these sources

References

Coolen F.P.A. and van der Laan P. (2001). Imprecise predictive selection based on low structure assumptions. Journal of Statistical Planning and Inference, 98(1-2), 259â€“277.
