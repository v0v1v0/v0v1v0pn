Phae.long.est R Documentation

Extended selection table of Phaethornis longirostris songs

Description

'Phae.long.est' contains an extented selection table following the warbleR package object format. The data contains the annotations and wave objects of 50 long-billed hermit songs from 10 different song types (5 per song type, 'lek.song.type' column) recorded at 8 leks ('lek' column) in the Caribbean slope of Costa Rica. THIS DATA SET HAS BEEN RENAMED AS 'lbh.est'. 'Phae.long.est' WILL BE DEPRECATED IN FUTURE VERSIONS.

Usage

data(Phae.long.est)

Format

One object of class extended selection table (following warbleR package format) and data frame.

Details

Extended selection table of Phaethornis longirostris (Long-billed Hermit) songs.
