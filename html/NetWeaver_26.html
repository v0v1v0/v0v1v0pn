rc.track.pos R Documentation

Get Track Positions

Description

Calculate inner and outer radius of a track.

Usage

rc.track.pos(track.id)

Arguments

track.id

integer, number of the track from the outermost.

Value

A vector with two radius values: out.pos and in.pos.
