addToSummary R Documentation

Add elements to the "Summary" subsection in the "Overview" section of a standard report.

Description

Add elements to the "Summary" subsection in the "Overview" section of a standard report.

Usage

  addToSummary(report, ...)

Arguments

report

Report element.
...

Elements that will be added to the overview section. Often these are paragraph elements.

Value

Updated report element.
