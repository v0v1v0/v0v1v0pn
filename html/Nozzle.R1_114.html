setMaintainerEmail R Documentation

Set email address of maintainer of report.

Description

Set email address of maintainer of report.

Usage

  setMaintainerEmail(report, ...)

Arguments

report

Report element.
...

One or more strings that will be concatenated to form the email address of the maintainer.

Value

Updated report element.
