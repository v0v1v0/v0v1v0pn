setSoftwareVersion R Documentation

Set the name of the software that used Nozzle to generate report, e.g. "Version 1.2".

Description

Set the name of the software that used Nozzle to generate report, e.g. "Version 1.2".

Usage

  setSoftwareVersion(report, ...)

Arguments

report

Report element.
...

One or more strings that will be concatenated to form the version of the software.

Value

Updated report element.
