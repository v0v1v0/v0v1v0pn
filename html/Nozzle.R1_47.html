getLogo R Documentation

Get logo file for one of six positions (three at the top, three at the bottom) in report.

Description

Get logo file for one of six positions (three at the top, three at the bottom) in report.

Usage

  getLogo(report, position)

Arguments

report

Report element.
position

One of LOGO.TOP.LEFT, LOGO.TOP.CENTER, LOGO.TOP.RIGHT, LOGO.BOTTOM.LEFT, LOGO.BOTTOM.CENTER, LOGO.BOTTOM.RIGHT.

Value

Path or URL to the logo file at position.
