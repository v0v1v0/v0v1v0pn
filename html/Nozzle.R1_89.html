newWebCitation R Documentation

Create a citation element that represents a document published online. This is a convenience wrapper for newCitation.

Description

Create a citation element that represents a document published online. This is a convenience wrapper for newCitation.

Usage

  newWebCitation(authors, title, url)

Arguments

authors

Names of authors.
title

Title of the document.
url

URL of the document.

Value

New element.
