setCopyright R Documentation

Set copyright messsage for report.

Description

Set copyright messsage for report.

Usage

  setCopyright(report, owner, year, statement = NA,
    url = NA)

Arguments

report

Report element.
owner

Copyright owner, e.g. "The President and Fellows of Harvard College".
year

Copyright year, e.g. "2012" or "2011-2013".
statement

Copyright statement, e.g. "All rights reserved.".
url

A URL that will be linked to the copyright owner.

Value

Updated report element.
