rle_encode_int R Documentation

RLE encode integers

Description

RLE encode integers

Usage

rle_encode_int(x)

Arguments

x

Integer vector.

Value

Raw vector, the encoded integers. It has two attributes:

  • bit_length: the number of bits needed to encode the input, and

  • length: length of the original integer input.
