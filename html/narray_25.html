rep R Documentation

Repeats an array along an arbitrary axis

Description

Repeats an array along an arbitrary axis

Usage

rep(x, n, along = 1)

crep(x, n)

rrep(x, n)

Arguments

x

An array object
n

Integer, how often to repeat
along

Along which axis to repeat (default: 1)

Value

An array that is repeated 'n' times on axis 'along'
