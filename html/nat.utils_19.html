Verify integrity of one or more zip files

Description

Verify integrity of one or more zip files

Usage

zipok(f, Verbose = FALSE)

Arguments

f

Path to one (or more) files
Verbose

Whether to be Verbose (default FALSE)

Details

Uses system unzip command.

Value

TRUE when file OK, FALSE otherwise
