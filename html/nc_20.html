melt_list R Documentation

melt list

Description

Compute a list of arguments to pass to melt.data.table.

Usage

melt_list(measure.fun, 
    dot.args, ...)

Arguments

measure.fun

measure_single or measure_multiple.
dot.args

list of arguments for check_df_names.
...

passed to measure.fun.
