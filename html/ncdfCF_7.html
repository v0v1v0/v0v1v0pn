CFAxisDiscrete R Documentation

Find indices in the axis domain

Description

This class represent discrete CF axes, i.e. those axes whose coordinate values do not represent a physical or categorical property. The coordinate values are ordinal values equal to the index into the axis.

Method indexOf()

Given a vector of numerical values x, find their indices in the values of the axis. In effect, this returns index values into the axis, but outside values will be dropped.

Usage
CFAxisDiscrete$indexOf(x, method = "constant")
Arguments
x

Vector of numeric values to find axis indices for.

method

Ignored.

Returns

Numeric vector of the same length as x. Values of x outside of the range of the values in the axis are returned as 0 and .Machine$integer.max, respectively.
