CFAxisLatitude R Documentation

Latitude CF axis object

Description

This class represents a latitude axis. Its values are numeric. This class adds some logic that is specific to latitudes, such as their range, orientation and their meaning.

Method sub_axis()

Return an axis spanning a smaller dimension range.

This method returns an axis which spans the range of indices given by the rng argument.

Usage
CFAxisLatitude$sub_axis(group, rng = NULL)
Arguments
group

The group to create the new axis in.

rng

The range of values from this axis to include in the returned axis.

Returns

A CFAxisLatitude covering the indicated range of indices. If the rng argument includes only a single value, an CFAxisScalar instance is returned with the value from this axis. If the value of the argument is NULL, return the entire axis (possibly as a scalar axis).
