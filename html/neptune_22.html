neptune_print_structure R Documentation

Pretty prints the structure of the run's metadata. Paths are ordered lexicographically and the whole structure is neatly colored.

Description

Pretty prints the structure of the run's metadata. Paths are ordered lexicographically and the whole structure is neatly colored.

Usage

neptune_print_structure(run)

Arguments

run

Neptune run object.

Value

None
