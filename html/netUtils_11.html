graph_cartesian R Documentation

Cartesian product of two graphs

Description

Compute the Cartesian product of two graphs

Usage

graph_cartesian(g, h)

Arguments

g

An igraph object
h

An igraph object

Details

See https://en.wikipedia.org/wiki/Cartesian_product_of_graphs

Value

Cartesian product as igraph object

Author(s)

David Schoch

Examples

library(igraph)
g <- make_ring(4)
h <- make_full_graph(2)
graph_cartesian(g, h)
