graph_direct R Documentation

Direct product of two graphs

Description

Compute the direct product of two graphs

Usage

graph_direct(g, h)

Arguments

g

An igraph object
h

An igraph object

Details

See https://en.wikipedia.org/wiki/Tensor_product_of_graphs

Value

Direct product as igraph object

Author(s)

David Schoch

Examples

library(igraph)
g <- make_ring(4)
h <- make_full_graph(2)
graph_direct(g, h)
