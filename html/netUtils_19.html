graph_kpartite R Documentation

k partite graphs

Description

Create a random k-partite graph.

Usage

graph_kpartite(n = 10, grp = c(5, 5))

Arguments

n

number of nodes
grp

vector of partition sizes

Value

igraph object

Author(s)

David Schoch

Examples

# 3-partite graph with equal sized groups
graph_kpartite(n = 15, grp = c(5, 5, 5))
