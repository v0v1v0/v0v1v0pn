clique_vertex_mat R Documentation

Clique Vertex Matrix

Description

Creates the clique vertex matrix with entries (i,j) equal to one if node j is in clique i

Usage

clique_vertex_mat(g)

Arguments

g

An igraph object

Value

Numeric matrix

Author(s)

David Schoch

Examples

library(igraph)
g <- sample_gnp(10, 0.2)
clique_vertex_mat(g)
