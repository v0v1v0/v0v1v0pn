get_netmhc2pan_archive_filename R Documentation

Get the filename of the NetMHC2pan archive file

Description

Get the filename of the NetMHC2pan archive file

Usage

get_netmhc2pan_archive_filename(
  netmhc2pan_version = get_default_netmhc2pan_version()
)

Arguments

netmhc2pan_version

the NetMHCIIpan version, for example 3.2. Use get_default_netmhc2pan_version to get the default NetMHCIIpan version

Value

the filename of the NetMHC2pan archive file

Examples

get_netmhc2pan_archive_filename()
