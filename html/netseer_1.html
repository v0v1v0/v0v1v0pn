generate_graph R Documentation

Generates a bigger graph by modifying an initial graph.

Description

Generates a bigger graph using parameters for node and edge growth.

Usage

generate_graph(gr = NULL, del_edge = 0.1, new_nodes = 0.1, edge_increase = 0.1)

Arguments

gr

The input graph to generate the next graph. If set to NULL a graph using igraph::sample_pa is used as the input graph.
del_edge

The proportion of edges deleted from the input graph. Default set to 0.1.
new_nodes

The proportion of nodes added to the input graph. Default set to 0.1.
edge_increase

The proportion of edges added to the input graph. Default set to 0.1.

Value

A graph.

Examples

set.seed(1)
gr <- generate_graph()
gr
