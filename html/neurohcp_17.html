hcp_ids R Documentation

Get list of HCP Ids

Description

Returns list of IDs of folders in the HCP database

Usage

hcp_ids(group = c("HCP", "HCP_900", "HCP_1200", "HCP_Retest"))

Arguments

group

Group of IDs to return

Value

Character vector

Examples

if (have_aws_key()) {
head(hcp_ids("HCP"))
head(hcp_ids("HCP_900"))
}
