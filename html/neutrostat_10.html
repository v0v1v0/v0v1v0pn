ncv R Documentation

CV of the neutrosophic data

Description

Neutrosophic coefficient of variation is an interval value of the neutrosphic numbers

Usage

ncv(data)

Arguments

data

data is a list of neutrosophic numbers

Value

Interval cv value.

Author(s)

Zahid Khan
