nic_palettes R Documentation

Nature Inspired Colour Palatte Package

Description

This is a collection of color palettes for R inspired by leaves. This work is inspired by the pnwcolours package in R and codes are modifies using pnwcolours. Create colour palettes Names: applecroton, colleasa, colleasb Type names(nic_palettes) to view the names of colour palattes

Usage

nic_palettes

Format

An object of class list of length 15.
