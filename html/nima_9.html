lm_plot R Documentation

Linear Model Diagnostic Plots

Description

Produce standard diagnostic plots for linear models using ggplot2.

Usage

lm_plot(x, ...)

Arguments

x

A linear model object produced by lm().
...

Extra arguments, currently ignored.

Examples

n <- 100
x1 <- rnorm(n)
y1 <- rnorm(n)
linmod <- lm(y1 ~ x1)
plot(linmod)
