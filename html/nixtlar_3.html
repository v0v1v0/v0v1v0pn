Description

Get NIXTLA_API_KEY from options or from .Renviron This is a private function of 'nixtlar'

Usage

.get_api_key()

Value

If available, the NIXTLA_API_KEY. Otherwise it returns an error and a message asking the user to set the 'API' key.
