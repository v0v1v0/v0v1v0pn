thin.default R Documentation

Thin MCMC Object

Description

Thins an MCMC object's iterations.

Usage

## Default S3 method:
thin(x, nthin = 1L, ...)

Arguments

x

An object.
nthin

A positive integer of the thinning rate.
...

Unused.

Value

The thinned MCMC object.

Examples

thin(nlists(nlist(x = 1), nlist(x = 2), nlist(x = 3), nlist(x = 4)), nthin = 2)
