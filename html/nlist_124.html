Description

Turn an object into a tidy tibble

Usage

## S3 method for class 'nlists'
tidy(x, simplify = FALSE, ...)

Arguments

x

An object.
simplify

A flag specifying whether to drop sd and zscore columns.
...

Unused.

Value

A tibble::tibble() with information about model components.
