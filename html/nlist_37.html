chk_nlist R Documentation

Check nlist Object or nlists Object

Description

chk_nlist checks if an nlist-object().

Usage

chk_nlist(x, x_name = NULL)

chk_nlists(x, x_name = NULL)

Arguments

x

The object to check.
x_name

A string of the name of object x or NULL.

Value

NULL, invisibly. Called for the side effect of throwing an error if the condition is not met.

Functions

  • chk_nlists: Check nlists Object

    chk_nlists checks if an nlists-object().

Examples


# chk_nlist
chk_nlist(nlist(x = 1))
try(chk_nlist(list(x = 1)))

# chk_nlists
chk_nlists(nlists(nlist(x = 1)))
