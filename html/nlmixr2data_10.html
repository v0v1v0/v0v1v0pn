mavoglurant R Documentation

Mavoglurant PK data

Description

This was used in a full PBPK model. This one was published for mavoglurant (Wendling et al. 2016).

Usage

mavoglurant

Format

A data frame with 2,678 rows by 14 columns

ID

Subject ID

CMT

Compartment Number

EVID

Event ID

MDV

Missing DV

DV

Dependent Variable, Mavoglurant

AMT

Dose Amount Keyword

TIME

Time (hr)

DOSE

Dose

OCC

Occasion

RATE

Rate

AGE

Age

SEX

Sex

WT

Weight

HT

Height

Source

Wendling et al. 2016
