pnlslm R Documentation

pnlslm

Description

Compact display of specified nls.lm object x. This code returns the iteration count in a different variable from that of nls objects.

Usage

pnlslm(x)

Arguments

x

an nls() result object from minpack.lm::nls.lm()

Value

none
