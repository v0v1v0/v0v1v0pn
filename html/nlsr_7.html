isCALL R Documentation

isCALL Test if argument is a particular call

Description

Test if x is a call to the function given by name.

Used in newSimplification definitions.

Usage

isCALL(x, name)

Arguments

x

object to be tested
name

The function to test for.

Examples

x <- quote(mean(1:10))
isCALL(x, "mean")
isCALL(x, "sd")
