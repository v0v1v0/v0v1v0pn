Sigmoid R Documentation

: Sigmoid

Description

The Sigmoid activation function.

Usage

Sigmoid(x)

Arguments

x

a real number that is the input to our Sigmoid function.

Value

The output of a standard Sigmoid function, i,e. \frac{1}{1 + \exp(-x)}. See also Tanh.and ReLU.

Examples

Sigmoid(0)
Sigmoid(-1)
