nms_nonmem.nms_pmx R Documentation

Convert pmxTools Names to NONMEM

Description

Converts nms_pmx to nms_nonmem. I.e., THETA1 stays THETA1, but OM1,1 becomes OMEGA(1,1).

Usage

## S3 method for class 'nms_pmx'
nms_nonmem(x, ...)

Arguments

x

nms_pmx
...

ignored

Value

nms_nonmem (character)
