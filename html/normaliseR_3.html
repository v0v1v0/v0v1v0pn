minmax_scaler R Documentation

Rescales a numeric vector into the unit interval [0,1]

Description

z_{i} = \frac{x_{i} - \text{min}(\mathbf{x})}{\text{max}(\mathbf{x}) - \text{min}(\mathbf{x})}

Usage

minmax_scaler(x)

Arguments

x

numeric vector

Value

numeric vector
