print.npc R Documentation

Print the npc object.

Description

Print the npc object.

Usage

## S3 method for class 'npc'
print(x, ...)

Arguments

x

fitted npc object using npc.
...

additional arguments.

See Also

npc, nproc

Examples

n = 1000
x = matrix(rnorm(n*2),n,2)
c = 1+3*x[,1]
y = rbinom(n,1,1/(1+exp(-c)))
fit = npc(x, y, method = 'lda')
print(fit)
