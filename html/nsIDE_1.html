copy_to_clipboard R Documentation

Copy Object or Syntax to Clipboard

Description

Copies the specified object or its syntax (as dput) to the clipboard, allowing for further use in external tools or environments.

Usage

copy_to_clipboard(object = NULL, object_name = NULL, type = "object")

Arguments

object

An R object to be copied. Either object or object_name must be provided.
object_name

A string specifying the name of an object in the global environment to be copied. Either object or object_name must be provided.
type

A string indicating the type of copy operation. Can be "object" (default) for copying the object content or "dput" for copying the dput representation of the object.

Value

Invisibly returns a JSON representation of the operation performed.

Examples

# Example 1: Copying an object directly
my_data <- data.frame(x = 1:5, y = letters[1:5])
copy_to_clipboard(my_data)

# Example 2: Copying an object by its name
copy_to_clipboard(object_name = "my_data")

# Example 3: Copying an object's `dput` syntax
copy_to_clipboard(my_data, type = "dput")
