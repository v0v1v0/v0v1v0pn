load_image R Documentation

Load an R Workspace

Description

Loads a saved R workspace from a specified file into the global environment.

Usage

load_image(save_path)

Arguments

save_path

A string specifying the file path of the saved workspace file (typically with a .RData extension).

Value

Invisibly returns TRUE upon successful loading of the workspace.

Examples

# Example 1: Load a previously saved workspace
if(F){
load_image("workspace_backup.RData")
}

# Example 2: Load a workspace as part of a script
if(F){
if (file.exists("backup_2023-12-01.RData")) {
  load_image("backup_2023-12-01.RData")
}
}
