load_lesson R Documentation

Load a Lesson

Description

Loads a lesson from a specified file path. Supports both .RData files and YAML-based lesson files, converting the latter into a course list format.

Usage

load_lesson(lesson_path)

Arguments

lesson_path

A string specifying the file path of the lesson to be loaded. This can be a .RData file for saved lessons or a YAML file for raw lesson definitions.

Value

Invisibly returns a JSON object containing:

  • action: The action performed (e.g., "load_lesson").

  • object_name: The modified course list without answers.

  • block_order: The order of blocks in the lesson.

  • lesson_path: The path of the loaded lesson.

  • current_lesson_item: The current lesson item index.

Examples

if(F){
# Example 1: Load a saved lesson from an RData file
load_lesson("lesson_progress.RData")

# Example 2: Load a raw lesson from a YAML file
load_lesson("lesson_definition.yaml")
}
