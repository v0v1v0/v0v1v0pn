save_image R Documentation

Save the Current R Workspace

Description

Saves the entire current R workspace, including all objects, to a specified file path.

Usage

save_image(save_path)

Arguments

save_path

A string specifying the file path where the workspace should be saved. The file is typically saved with a .RData extension.

Value

Invisibly returns TRUE upon successful saving of the workspace.

Examples

# Example 1: Save the current workspace to a specific file
if(F){
save_image("workspace_backup.RData")
}

# Example 2: Automate workspace saving in a script
if(F){
save_image(paste0("backup_", Sys.Date(), ".RData"))
}
