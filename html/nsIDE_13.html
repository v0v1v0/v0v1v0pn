save_lesson R Documentation

Save a Lesson and Workspace

Description

Saves the current lesson object and the entire R workspace to a specified file path.

Usage

save_lesson(current_lesson_item, save_path)

Arguments

current_lesson_item

An object representing the current lesson item to be saved. This is typically a structured object containing lesson data.
save_path

A string specifying the file path where the lesson and workspace should be saved. Commonly, this file should have a .RData extension.

Value

Invisibly returns TRUE upon successful saving of the lesson and workspace.

Examples

if(F){
# Example 1: Save a lesson and workspace to a file
lesson <- 1
save_lesson(lesson, "advanced_r_lesson.RData")

# Example 2: Automate lesson saving
current_lesson <- 1
save_lesson(current_lesson, paste0("lesson_", Sys.Date(), ".RData"))
}
