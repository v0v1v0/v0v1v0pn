test_lesson_block R Documentation

Test a Lesson Block

Description

Evaluates a specific lesson block by checking if the variables in the global environment match the expected answers defined in the block.

Usage

test_lesson_block(block_name)

Arguments

block_name

A string specifying the name of the lesson block to be tested. The block should exist within the current lesson object (.u____curious_lesson).

Value

A character vector with two elements:

  • The first element is "TRUE" or "FALSE" indicating whether the test passed.

  • The second element is a message providing feedback or details about the test results.

Examples

if(F){
# Example 1: Test a lesson block by name
test_lesson_block("block1")

# Example 2: Handle the results of the test
result <- test_lesson_block("block2")
if (result[1] == "TRUE") {
  cat("Block passed successfully!\n")
} else {
  cat("Block failed: ", result[2], "\n")
}
}
