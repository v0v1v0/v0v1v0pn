text_to_syntax R Documentation

Convert Text to R Syntax

Description

Converts a tab-separated text input into R syntax using dput. Supports both single-column and multi-column tabular data.

Usage

text_to_syntax(text)

Arguments

text

A string containing tab-separated data. The text can represent a single column or a tabular dataset with multiple columns.

Value

A character vector containing the formatted R syntax representation of the input data.

Examples

# Example 1: Convert single-column text to R syntax
text <- "1\n2\n3\n4"
syntax <- text_to_syntax(text)
cat(syntax, sep = "\n")

# Example 2: Convert multi-column tab-separated text to R syntax
text <- "a\tb\n1\t2\n3\t4"
syntax <- text_to_syntax(text)
cat(syntax, sep = "\n")
