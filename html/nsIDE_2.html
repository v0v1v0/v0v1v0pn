create_zip R Documentation

Create a ZIP Archive for Project Files

Description

Creates a ZIP archive containing selected directories and files associated with a project, such as scripts, libraries, and data.

Usage

create_zip(zip_name)

Arguments

zip_name

A string specifying the name of the ZIP file to be created. Special characters like "-" or ":" will be removed.

Value

Returns TRUE invisibly upon successful creation of the ZIP file.

Examples

# Example 1: Create a ZIP archive for the current project
if(F){
create_zip("project_backup_20231209")
}
