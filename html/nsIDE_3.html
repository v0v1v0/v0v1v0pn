download_course R Documentation

Download and Extract Course Content

Description

Downloads a specific section of a course as a ZIP file and extracts its contents into a designated directory.

Usage

download_course(course_name, section_name)

Arguments

course_name

A string specifying the name of the course to download.
section_name

A string specifying the name of the course section to download.

Value

Returns the result of the extraction process, typically the list of files extracted, or NULL if the download fails.

Examples

# Example 1: Download and extract a specific course section
if(F){
download_course("Learning-How-to-Program", "Statements-and-Variables")
}

# Example 2: Automate downloading a batch of sections
if(F){
sections <- c("Statements-and-Variables", "Working-with-Vectors")
lapply(sections, function(sec) download_course("Learning-How-to-Program", sec))
}
