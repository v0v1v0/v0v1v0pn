extract_zip R Documentation

Extract ZIP File and Process Metadata

Description

Extracts the contents of a specified ZIP file and processes metadata for selected files, such as scripts and README files.

Usage

extract_zip(zip_name)

Arguments

zip_name

A string specifying the name or path of the ZIP file to be extracted.

Value

Returns TRUE invisibly upon successful extraction. Generates and logs a JSON file containing metadata about the extracted files.

Examples

# Example 1: Extract a ZIP file and process metadata
if(F){
extract_zip("course_material.zip")
}

# Example 2: Use within a workflow to extract and validate files
if(F){
zip_file <- "data_archive.zip"
if (file.exists(zip_file)) {
  extract_zip(zip_file)
}
}
