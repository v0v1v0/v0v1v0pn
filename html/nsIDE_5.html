file_info R Documentation

Retrieve File Information

Description

Retrieves detailed information about a specified file, including its name, path, size, and last modified timestamp.

Usage

file_info(file_path)

Arguments

file_path

A string specifying the path to the file for which information is to be retrieved.

Value

Returns a list containing the file's name, path, size (in KB), and last modified timestamp. If the file does not exist, a message is returned indicating the error.

Examples

# Example 1: Get information about an existing file
if(F){
 file_info("example.txt")
}

# Example 2: Handle non-existent file gracefully
if(F){
 file_info("non_existent_file.txt")
}
