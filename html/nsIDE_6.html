get_all_package_docs R Documentation

Retrieve Documentation for All Functions in a Package

Description

Collects and returns the documentation for all exported functions in a specified package.

Usage

get_all_package_docs(package_name)

Arguments

package_name

A string specifying the name of the package whose function documentation is to be retrieved.

Value

A named list where each element contains the documentation for a function in the specified package. The names of the list correspond to the function names.

Examples

# Example 1: Get documentation for functions in the `stats` package
docs <- get_all_package_docs("stats")
names(docs) # List all functions in the package
docs$lm # View documentation for the `lm` function
