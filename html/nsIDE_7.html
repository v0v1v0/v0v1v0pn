ide_session_information R Documentation

Retrieve IDE Session Information

Description

Collects detailed information about the current IDE session, including file structures, scripts, ZIP files, courses, R objects, and package details.

Usage

ide_session_information(environment_will_return_records)

Arguments

environment_will_return_records

A logical value. Determines whether metadata for R objects should be extracted from the provided environment or skipped.

Value

A JSON object containing:

  • files: Information about files in the ⁠in⁠, out, and mid directories.

  • scripts_df: Information about R script files in the scripts directory.

  • zips_df: Information about ZIP files in the zips directory.

  • courses_df: Information about course files in the courses directory.

  • objects: Metadata for R objects.

  • attached_packages: A list of currently attached packages.

  • detached_packages: A list of installed packages not currently attached.

Examples

# Example 1: Retrieve session information with environment metadata
if(F){
session_info <- ide_session_information(T)
}

# Example 2: Skip environment metadata extraction
if(F){
session_info <- ide_session_information(F)
}
