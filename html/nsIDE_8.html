ide_view R Documentation

View an Object in the IDE

Description

Formats and prepares an object for viewing in the IDE. Supports vectors, data frames, and matrices, with options for viewing as a dictionary.

Usage

ide_view(object = NULL, object_name = NULL, type = "object", ...)

Arguments

object

An R object to be viewed. Either object or object_name must be provided.
object_name

A string specifying the name of an object in the global environment to be viewed. Either object or object_name must be provided.
type

A string specifying the view type. Default is "object". Use "dictionary" for a tabular dictionary-style view.
...

Additional arguments passed to helper functions for formatting, such as util_create_table_simple.

Value

Invisibly returns a JSON object containing:

  • action: The action performed (e.g., "view").

  • object_name: The name of the object.

  • type: The view type.

  • return_object: The formatted data frame representation of the object.

Examples

# Example 1: View a vector
ide_view(c(1, 2, 3, 4))

# Example 2: View a data frame
df <- data.frame(x = 1:5, y = letters[1:5])
ide_view(df)

# Example 3: View an object by name
my_matrix <- matrix(1:9, nrow = 3)
ide_view(object_name = "my_matrix")

# Example 4: View a data frame as a dictionary
ide_view(df, type = "dictionary")
