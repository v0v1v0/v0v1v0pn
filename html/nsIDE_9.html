install_packages R Documentation

Install R Packages with Dependencies

Description

Installs specified R packages and their dependencies from a custom package repository. Ensures that only missing packages are downloaded and installed.

Usage

install_packages(packages)

Arguments

packages

A character vector of package names to install.

Value

Invisibly returns TRUE upon successful installation of all requested packages.

Examples

if(F){
# Example 1: Install a single package
install_packages("dplyr")

# Example 2: Install multiple packages
install_packages(c("ggplot2", "data.table"))

# Example 3: Ensure a package and its dependencies are installed
install_packages("caret")
}
