Description

Bhavcopy PR from NSE

Usage

bhavpr(x)

Arguments

x

numeric date format

Details

Gets Bhavcopy PR from NSE for the given date.

Value

List of Bhavcopy PR for the given date.

Note

The date should be strictly numerical and mentioned in quotation mark, refer example.
