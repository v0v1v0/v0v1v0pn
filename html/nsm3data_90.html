toc R Documentation

toc

Description

help(toc) displays the table of contents for datasets.

Usage

toc()

Details

A guide to the datasets in nsm3data. Datasets are listed under the Arguments section by the order they appear in the Nonparametric Statistical Methods textbook, 3ed. The reference name in the nsm3data package is given, along with the table number and the descriptor of the dataset.
