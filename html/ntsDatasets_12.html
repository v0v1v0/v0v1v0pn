NOx R Documentation

Nitrogen Oxides Emissions Data

Description

This dataset includes yearly nitrogen oxides emissions for Denmark for the period 1990 to 2018.

Format

A data frame with 29 observations of nitrogen oxides emissions for Denmark for the period 1990-2018.

Source

Khan, Z., Amin, A., Khan, S. A., Gulistan, M. (2021). Statistical Development of the Neutrosophic Lognormal Model with Application to Environmental Data, Neutrosophic Sets and Systems, 47, 1-11.

Examples

data("NOx")
