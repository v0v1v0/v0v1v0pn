PopVillUSA R Documentation

Population Villages USA data

Description

It is related to population compactness of a few villages in rural USA.

Format

A data frame with 17 populations of Villages in the USA. Each row represents a village.

Source

Albassam, M., Khan, N., Aslam, M., (2020) The W/S Test for Data Having Neutrosophic Numbers: An Application to USA Village Population, Complexity, 2020, 3690879.

Rao, G. S., Norouzirad, M., and Mazarei, D. (2023). Neutrosophic Generalized Exponential Distribution with Application. Neutrosophic Sets and Systems, 55, 471-485.

Examples

data("PopVillUSA")
