debt R Documentation

Public Debt Data

Description

This dataset contains information on public debt (in million dollars) in Egypt, covering the period from 2000 to 2022.

Format

A data frame with 23 observations of public debt in Egypt from 2000 to 2022.

Source

Essa, N. I., Zhar, H. M., Abu El-Magd, N. A. T. (2023). Neutrosophic Generalized Pareto Distribution, Mathematics and Statistics, 11(5), 827-833.

Examples

data("debt")
