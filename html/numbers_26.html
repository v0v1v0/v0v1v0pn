iNthroot R Documentation

Integer N-th Root

Description

Determine the integer n-th root of .

Usage

  iNthroot(p, n)

Arguments

p

any positive number.
n

a natural number.

Details

Calculates the highest natural number below the n-th root of p in a more integer based way than simply floor(p^{1/n}).

Value

An integer.

Examples

iNthroot(0.5, 6)    # 0
iNthroot(1, 6)      # 1
iNthroot(5^6, 6)    # 5
iNthroot(5^6-1, 6)  # 4
