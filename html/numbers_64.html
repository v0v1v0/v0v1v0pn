Quadratic Residues

Description

List all quadratic residues of an integer.

Usage

quadratic_residues(n)

Arguments

n

integer.

Details

Squares all numbers between 0 and n/2 and generate a unique list of all these numbers modulo n.

Value

Vector of integers.
