f_ordinal R Documentation

Add Ordinal Suffixes (-st, -nd, -rd, -th) to Numbers

Description

Add ordinal suffixes (-st, -nd, -rd, -th) to numbers.

Usage

f_ordinal(x, ...)

ff_ordinal(...)

Arguments

x

A vector of numbers (or string equivalents).
...

ignored.

Value

Returns a string vector with ordinal suffixes.

Examples

f_ordinal(1:25)
