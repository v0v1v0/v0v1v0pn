f_state R Documentation

Format State Names as Abbreviations

Description

Formats a state name as the abbreviated form.

Usage

f_state(x, ...)

ff_state(...)

Arguments

x

A vector of states.
...

ignored.

Value

Returns a string of abbreviated states.

Examples

f_state(c('Texas', 'New York', NA, 'New Jersey', 'Washington', 'Europe'))
