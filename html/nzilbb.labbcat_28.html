getGraphIds R Documentation

Deprecated synonym for getTranscriptIds.

Description

Returns a list of graph IDs (i.e. transcript names).

Usage

getGraphIds(labbcat.url)

Arguments

labbcat.url

URL to the LaBB-CAT instance

Value

A list of graph IDs

getTranscriptIds
